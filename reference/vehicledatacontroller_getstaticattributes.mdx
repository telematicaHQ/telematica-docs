---
title: "Static Attributes"
description: "Get the static attributes of the vehicle"
api:
  "GET https://api.telematica.so/v2/vehicle-data/static-attributes/{vehicleId}"
---

### Path Params

<ParamField path="vehicleId" type="string" required>
  24-char hex identifier for the vehicle. The owner must have provided consent
  through Telematica consent flow.
</ParamField>

### Response

<ParamField body="vin" type="string" required />

<ParamField body="model" type="string" required />

<ParamField body="year" type="number" required />

<ParamField body="make" type="string" required />

<ParamField body="registration" type="string" required />

<RequestExample>

```bash Request
curl --request GET \
     --url https://api.telematica.so/v2/vehicle-data/static-attributes/vehicleId \
     --header 'accept: application/json'
```

</RequestExample>
