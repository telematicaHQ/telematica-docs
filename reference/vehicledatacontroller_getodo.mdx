---
title: "Odometer"
description: "Get the current odometer reading of the vehicle in kms"
api: "GET https://api.telematica.so/v2/vehicle-data/odo/{vehicleId}"
---

### Path Params

<ParamField path="vehicleId" type="string" required>
  24-char hex identifier for the vehicle. The owner must have provided consent
  through Telematica consent flow.
</ParamField>

### Response

<ParamField body="odo" type="number" required>
  The current odomerter reading in km
</ParamField>

<ParamField body="timestamp" type="number" required>
  unix timestamp in milliseconds, time when data was fetched from OEM servers by
  Telematica
</ParamField>

<RequestExample>

```bash Request
curl --request GET \
     --url https://api.telematica.so/v2/vehicle-data/odo/vehicleId \
     --header 'accept: application/json'
```

</RequestExample>
