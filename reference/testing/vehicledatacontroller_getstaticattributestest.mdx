---
title: "Test static-attributes"
description: "Test the static-attributes endpoint on a dummy vehicle"
api: "GET https://api.telematica.so/v1/vehicle-data/static-attributes/test"
---

You just need a valid token but not a valid vehicleId to test these endpoints.
(You can use the example ids by clicking on input field and selecting example
input)

They will return dummy data in the same format as of Vehicle Data endpoints

### Query Params

<ParamField query="vehicleId" type="string" required>
  24-char hex identifier for the vehicle. The owner must have provided consent
  through Telematica consent flow.
</ParamField>

### Response

<ParamField body="vin" type="string" required />

<ParamField body="model" type="string" required />

<ParamField body="year" type="number" required />

<ParamField body="make" type="string" required />

<ParamField body="registration" type="string" required />

<RequestExample>

```bash Request
curl --request GET \
     --url https://api.telematica.so/v1/vehicle-data/static-attributes/test \
     --header 'accept: application/json'
```

</RequestExample>
