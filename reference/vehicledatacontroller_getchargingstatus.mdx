---
title: "Charging Status"
description: "Get the current charging status of the vehicle"
api: "GET https://api.telematica.so/v2/vehicle-data/charging-status/{vehicleId}"
---

### Path Params

<ParamField path="vehicleId" type="string" required>
  24-char hex identifier for the vehicle. The owner must have provided consent
  through Telematica consent flow.
</ParamField>

### Response

<ParamField body="isCharging" type="boolean" required />

<ParamField body="timestamp" type="number" required>
  unix timestamp in milliseconds, time when data was fetched from OEM servers by
  Telematica
</ParamField>

<RequestExample>

```bash Request
curl --request GET \
     --url https://api.telematica.so/v2/vehicle-data/charging-status/vehicleId \
     --header 'accept: application/json'
```

</RequestExample>
