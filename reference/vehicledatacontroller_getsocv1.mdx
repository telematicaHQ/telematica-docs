---
title: "SOC (battery level)"
description: "Get the current SOC (battery level) of the vehicle"
api: "GET https://api.telematica.so/v1/vehicle-data/soc"
---

### Query Params

<ParamField query="vehicleId" type="string" required>
  24-char hex identifier for the vehicle. The owner must have provided consent
  through Telematica consent flow.
</ParamField>

### Response

<ParamField body="soc" type="number" required>
  State Of Charge of the vehicle (percentage value)
</ParamField>

<ParamField body="timestamp" type="number" required>
  unix timestamp in milliseconds, time when data was fetched from OEM servers by
  Telematica
</ParamField>

<RequestExample>

```bash Request
curl --request GET \
     --url https://api.telematica.so/v1/vehicle-data/soc \
     --header 'accept: application/json'
```

</RequestExample>
