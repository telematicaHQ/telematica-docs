#### **Can I secure my POST endpoint with a bearer token?**

Yes, please make sure you communicate the details over
[email](mailto:aditya@telematica.so) if you choose to.

#### Can we skip the brand selection screen in consent flow?

Yes, just add a query parameter to the consent UI url as shown below:

```bash
https://us.telematica.so/connect?app=<app-id>&user=<user_id_in_your_db>&brand=Tata
```
